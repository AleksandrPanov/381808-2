Требования к входным/выходным данным
input:
    Polynom_1
    Polynom_2
    x
output:
    Polynom_1 + Polynom_2 = ...
    Polynom_1 - Polynom_2 = ...
    Polynom_1 * Polynom_2 = ...
    Polynom_1(x) = ...
    Polynom_2(x) = ...
    
Требования к реализации:
класс Polynom состоит из списка элементов класса/структуры Monom
class Monom
{
    double coeff;
    int power;
}

класс Polynom должен иметь:
конструктор копирования
операторы + - * =

список мономов должен быть реалзиван с помощью собственного класса List
класс List должен иметь:
конструктор копирования
методы для вставки/удаления элементов
деструктор

Особенности полиномов:
1. В полиноме не должно быть нескольких мономов с одинаковыми степенями (после умножения двух полиномов, нужно сокращать мономы с одинаковыми степенями).
2. В полиноме не может быть мономов с коэффициентом ноль.
